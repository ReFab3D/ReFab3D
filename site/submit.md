# Proposer une pièce introuvable

Vous pouvez proposer une pièce uniquement via GitHub.

## Conditions
- La pièce ne doit plus être fabriquée officiellement
- Vous fournissez des photos ou documents exploitables
- Vous acceptez une éventuelle commercialisation

---

## Étapes
1. Créez une issue GitHub via le formulaire dédié
2. La demande est analysée
3. Vous êtes tenu informé via GitHub

---

➡️ [Soumettre une pièce sur GitHub](https://github.com/<ton-username>/ReFab3D/issues/new/choose)

⚠️ ReFab3D se réserve le droit de refuser toute demande.

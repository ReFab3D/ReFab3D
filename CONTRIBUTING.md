## Proposer une pièce – ReFab3D

Les propositions se font exclusivement via une issue GitHub dédiée.

Aucune garantie de validation n’est donnée.

Conditions minimales :
- La pièce ne doit plus être fabriquée officiellement
- La pièce doit être techniquement reproductible
- Le demandeur accepte une éventuelle commercialisation

Toute proposition incomplète sera ignorée.
